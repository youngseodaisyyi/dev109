Bellevue College

Refelctions on SQL Fundamentals (BUSIT 103): I just knew the very basics of SQL commands like SELECT, FROM, and ORDER BY. I learned different commands such as GROUP BY, HAVING, INNER JOIN, etc. Throughout this class, I was supposed to learn the fundamentals of database structure, which I did. I also learn techniques useful for querying databases and learned to apply their skills in realistic scenarios extracting data and organizing it into meaningful information. I believed I gained experience with database servers and client tools, which can be applied in my future work.

Refelctions on Intro to Web Development (Dev 109): I did not know anything about html before starting this class. After taking this class, I learned how to create a hosting and domain account. I also learned how to use CSS, Javascript, etc. Some specific things I learned was to use Document Object Model to validate a form, create a slideshow using javascript, and refactoring.

Refelctions on Child Development (EDUC 115): I initially took this class because I have a lot people around me having babies or already with children. I thought that I could learn a little bit about how younger children develop, how they think, how to interact with them, etc. Throughout this class, I learned about theories regarding child development, different stages of pregnancies, growth of a child, pre-teen years, etc.
